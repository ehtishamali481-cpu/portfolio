## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/ehtishamali481-cpu/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```
